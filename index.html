如何使用範本

English version

善於利用範本功能可以讓您更方便而快速地使用 HackMD 書寫，包含以下三個優點：

  1. 為您節省每次新增類似格式的筆記耗費的時間
    （打開一篇新筆記、再打開舊的筆記、複製內容、再貼到新筆記、刪去要修改的部分…）
  2. 增加您筆記內容的一致性
    （可以將類似的筆記，如：部落格貼文、每日工作紀錄、會議記錄等，用相同樣式呈現。）
  3. 重複使用特定內容
    （筆記內如有共用的段落，如：常用圖示或常用清單，可以用插入範本的方式來實現。）

:::info
:bulb: 小提示： 如果您想要套用的模板主要是用 html <style> 自訂版型（例如：中文直書版型、或者黑色背景版型）的話，用嵌入筆記的方式實現更適合喔！（教學在此）

  • 中文直書的版型範例
  • 黑色背景的板型範例
    :::

使用範本

情境 1. 全新筆記套用範本：

方法一：

您可以將滑鼠懸停在新增筆記的按鈕 :heavy_plus_sign: 的上面，按鈕下方會出現 “從範本新增筆記” 的選項，點擊即可進入範本管理介面。於範本管理介面選擇您想要使用的範本，再點擊 “套用範本” 即可。

方法二：

在開啟新筆記以後，頁面下方會出現您最常用的前三個範本，點擊即可套用。

情境 2. 插入常用的重複內容：

點擊功能列的選單（），可以在筆記範本區塊中選擇 “插入範本”，點擊也會進入範本管理介面。當您從範本管理介面 “套用範本” 的時候，範本會在您此時游標的位置插入。
在您寫作的過程中，如果有經常需要重複使用的文字段落，可在將其儲存成自訂範本後，利用範本功能快速呼叫並重複使用。

自訂範本

方法一：

開啟一篇新筆記，輸入您想要存成範本的文字後，點擊功能列的選單，可以在筆記範本區塊中選擇 “存為範本” 後，再給予範本一個名稱即可。（下面示意圖中的黑色背景模板在此。）

方法二：

在範本管理介面右上方，點選 “建立新範本”，同樣給予範本一個名稱即可。
:::info
:bulb: 小提示： 用這個方法建立的新範本，其標題預設就會是範本名稱，但是直接修改標題並不會影響到範本的名稱喔！如果需要更改範本名稱，請閱讀管理範本。
:::

管理範本

範本的管理（選用、變更名稱、編輯、或者刪除）都在範本管理介面中進行。
:::info
:bulb: 小提示： 在管理介面的範本預覽是以筆記預覽的形式呈現，因此，在筆記預覽看不到的元素，如： YAML metadata、HTML 元素<style> 或被 <!-- 註解掉的文字段落 --> 等，就不會出現在管理介面的範本預覽喔！
:::

因為範本筆記其實是將一篇一般的筆記在資料庫裡標記為範本，所以當您想要刪除範本的時候，HackMD 會請您確認：您是否要將該筆記同時刪除，或者只是將其移出範本庫並轉為一般筆記？

:::danger
:warning: 警告： 反過來說，當您刪除 “被存為範本的筆記” 時，該範本也會自範本庫中移除喔！
:::

編輯範本和編輯一般筆記的方式一樣。對範本做的編輯也會即時儲存（影響）到範本庫內的範本，不需要另外手動儲存。如果您不慎編修到範本，也可以到 “修訂紀錄” 裡面回復原來的版本。

:::info
:bulb: 小提示： 在範本上做的編輯不會自動套用到之前套用範本的筆記喔！如果您想要這樣的效果的話，用嵌入筆記的方式更適合喔！
:::

總覽頁面

我們在 “總覽” 頁面也增加了單獨只顯示範本的選項。

另外，為了避免您不慎編輯到範本，範本的卡片右下角都有一個範本的符號，而且在您編輯範本的時候，編輯區塊的上方也會出現一個綠色橫幅提示您：您目前正在編輯的是範本而不是一般筆記。
